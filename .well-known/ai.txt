## Site Description
Eggcellent is a local-first AI prompt testing playground that helps developers test, debug, and compare outputs across different AI models. The platform supports multiple AI providers including OpenAI, Anthropic, Google, and xAI.

## Intended Use
- Primary: AI prompt testing and debugging
- Secondary: Model comparison and prompt version management
- Tertiary: Structured output validation

## Data Handling
- All API keys are stored locally in the browser
- No user data is sent to our servers
- Direct API calls to AI providers only
- Local-first architecture for privacy

## Key Features
1. Multi-model testing
2. Prompt version control
3. Local-first security
4. Structured validation
5. Multi-modal support
